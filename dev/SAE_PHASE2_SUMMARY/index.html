    How to Use Your SAE¶

    1. Evaluate Quality¶

    Run the comprehensive evaluation:

    # Make sure mlx_lm is installed
uv add mlx-lm

# Run evaluation
python examples/evaluate_sae.py

    This will output: - Reconstruction metrics (MSE, cosine similarity) - Sparsity metrics (L0, dead features %) - Feature interpretability (top examples per feature) - Quality score (0-5 scale)

    Results saved to: sae_layer23_mlp_10000samples_evaluation.json

    2. Analyze Features¶

    Find which features activate for specific texts:

    from mlx_lm import load
from mlxterp import InterpretableModel
from mlxterp.sae import BatchTopKSAE
from examples.sae_feature_analysis import get_top_activating_features

# Load SAE
sae = BatchTopKSAE.load("sae_layer23_mlp_10000samples.mlx")

# Load model
model, tokenizer = load("arogister/Qwen3-8B-ShiningValiant3-mlx-4Bit")
interp = InterpretableModel(model, tokenizer=tokenizer)

# Analyze a text
text = "Artificial intelligence is transforming technology"
top_features = get_top_activating_features(
    sae, interp, text,
    layer=23, component="mlp",
    top_k=10
)

for feat_id, activation in top_features:
    print(f"Feature {feat_id}: {activation:.4f}")

    Usage Example: Complete Workflow¶

    from mlx_lm import load
from mlxterp import InterpretableModel, SAEConfig
from mlxterp.sae import BatchTopKSAE
from datasets import load_dataset

# =============================================================================
# Step 1: Train SAE
# =============================================================================

model, tokenizer = load("mlx-community/Llama-3.2-1B-Instruct-4bit")
interp = InterpretableModel(model, tokenizer=tokenizer)

dataset = load_dataset("wikitext", "wikitext-2-raw-v1", split="train")
texts = [item["text"] for item in dataset if len(item["text"].strip()) > 50][:10000]

config = SAEConfig(
    sae_type="batchtopk",
    expansion_factor=32,
    k=128,
    learning_rate=3e-4,
    use_ghost_grads=True,
    num_epochs=3,
    use_wandb=True,
)

sae = interp.train_sae(
    layer=12,
    component="mlp",
    dataset=texts,
    config=config,
    save_path="my_sae.mlx"
)

# =============================================================================
# Step 2: Evaluate Quality
# =============================================================================

# Edit evaluate_sae.py to point to your SAE
# Then run: python examples/evaluate_sae.py

# =============================================================================
# Step 3: Analyze Features
# =============================================================================

from examples.sae_feature_analysis import (
    get_top_activating_features,
    get_top_activating_texts
)

# Find features for a text
text = "Machine learning is transforming AI research"
top_features = get_top_activating_features(
    sae, interp, text,
    layer=12, component="mlp",
    top_k=10
)

print(f"Top features for: '{text}'")
for feat_id, activation in top_features:
    print(f"  Feature {feat_id}: {activation:.4f}")

    # Find what this feature represents
    examples = get_top_activating_texts(
        sae, interp,
        feature_id=feat_id,
        texts=texts[:500],  # Search subset
        layer=12, component="mlp",
        top_k=5
    )

    print(f"  Top examples:")
    for ex_text, ex_act, pos in examples[:3]:
        print(f"    [{ex_act:.4f}] {ex_text[:80]}...")

    Faster Training (Optional)¶

    For smaller experiments, use pre-collection:

    from mlxterp.utils import get_activations

# Collect activations once (slow but one-time)
activations = get_activations(
    model, prompts=texts,
    layer=12, component="mlp"
)

# Train on pre-collected (80-200 it/s - 50-100x faster!)
sae = trainer.train(sae, activations)

    Troubleshooting¶

    If Evaluation Fails¶

    # Install missing dependency
uv add mlx-lm

# Run evaluation
python examples/evaluate_sae.py

    If Features Aren't Interpretable¶

    • Train longer (more epochs)
    • Use more diverse data
    • Increase dataset size
    • Try different layers

    If Dead Features > 80%¶

    • Your training stopped early (step ~200)
    • Ghost grads activate at step 1000+
    • Either:
    • Continue training to step 5000+
    • Retrain with dead_feature_window=500
